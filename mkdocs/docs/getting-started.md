## Установка

### Автоматическая установка (рекомендуется)

```bash
# клонирование и установка
chmod +x install.sh
./install.sh
```

Скрипт выполнит:
1. Проверку зависимостей (Docker, Docker Compose)
2. Генерацию паролей и секретов
3. Создание структуры директорий
4. Настройку схем базы данных
5. Сборку и запуск всех сервисов

С собственными параметрами:

```bash
./install.sh --db-password mypass --admin-password myadmin --production
```

### Ручная установка

```bash
# 1. Создать файл окружения
cp .env.example .env
nano .env

# 2. Создать директории
mkdir -p {logs,backup_DB,data,db/init,db/init-analytics}

# 3. Запустить базу данных
docker compose up -d postgres postgres-analytics redis

# 4. Дождаться инициализации (30 сек)
sleep 30

# 5. Запустить все сервисы
docker compose up -d
```

### Установка для продакшена

```bash
./install.sh \
  --production \
  --db-password "$(openssl rand -base64 32)" \
  --jwt-secret "$(openssl rand -hex 64)" \
  --admin-password "$(openssl rand -base64 16)"
```

## Первый вход

1. Откройте **http://localhost:15000**
2. Войдите с учётными данными по умолчанию:
   - **Имя пользователя:** `admin`
   - **Пароль:** `admin`
3. **Сразу смените пароль администратора!** Настройки → Профиль → Сменить пароль

## Интерфейс

**Сайдбар** — по умолчанию свёрнут, отображает только иконки. Нажмите кнопку ☰ в левом верхнем углу, чтобы развернуть полное меню.

- **Свёрнутый режим**: только иконки, ширина 64px
- **Развёрнутый режим**: иконки + названия, ширина 256px
- **Группы с подменю**: Торговля, Аналитика, Стратегии, Мониторинг — раскрываются по клику
- На мобильных устройствах сайдбар открывается поверх контента

Навигация переключается плавно — контент сдвигается вправо при раскрытии сайдбара.
