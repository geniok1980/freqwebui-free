# 💰 Финансовые данные (V7)

Финансовые данные — интеграция внешних финансовых котировок в FreqDash: криптовалюты, акции, новости и макроэкономические индикаторы.

## Сбор данных

Финансовые данные собираются автоматически через сервис **сборщика финансовых данных** (включён по умолчанию). Отключить можно установкой `DISABLE_FINANCE_COLLECTORS=true` в `.env`.

Данные хранятся в отдельной базе `financial_data` (в том же PostgreSQL).
