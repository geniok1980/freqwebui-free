# 🤖 Агентская стратегия (V8)

Агентская стратегия — система динамических торговых агентов, которая подстраивает веса торговых сигналов в зависимости от рыночного режима.

## Как это работает

1. Система определяет текущий **рыночный режим** (рост, падение, боковик, высокая волатильность)
2. Для каждого режима используются свои **веса сигналов**
3. Агенты обучаются на исторических данных, оптимизируя веса
4. Результаты сохраняются в базу `financial_data`

Сигналы, используемые в агентах:

| Сигнал | Описание |
|--------|----------|
| Моментум цены | Скорость изменения цены |
| Объём | Объём торгов |
| Тональность | Рыночные настроения (новости + соцсети) |
| Макро | Макроэкономические показатели |
| Стакан | Данные книги ордеров |

### Управление агентом через Docker

Система предоставляет управление контейнерами агентов:

- **Запустить контейнер** — запуск нового Docker-контейнера агента
- **Остановить / Запустить** — управление состоянием
- **Просмотр логов** — логи работы агента
- **Обновить** — обновление конфигурации агента
