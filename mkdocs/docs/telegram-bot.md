# 🤖 Telegram Bot

FreqDash включает Telegram-бота для мониторинга и управления ботами прямо из мессенджера.

## Настройка

### 1. Создайте бота в Telegram

Напишите [@BotFather](https://t.me/BotFather) и создайте нового бота:

```
/newbot
→ FreqDashBot
→ freqdash_bot
```

Сохраните полученный токен (вида `1234567890:ABCdefGHIjklmNOPqrstUVwxyz`).

### 2. Узнайте свой Telegram ID

Напишите [@userinfobot](https://t.me/userinfobot) — он пришлёт ваш числовой ID.

### 3. Введите настройки в веб-интерфейсе

1. Откройте **FreqDash → Настройки → Telegram Bot**
2. Введите **Токен бота** (от @BotFather)
3. Введите **Разрешённые пользователи** — ваш Telegram ID (можно несколько через запятую)
4. Нажмите **«Сохранить все»**

### 4. Запустите контейнер

```bash
docker compose up -d telegram-bot
```

Бот автоматически подхватит настройки из API.

## Изменение настроек без перезагрузки

Если вы изменили **токен** или **список пользователей** в веб-интерфейсе:

- **Разрешённые пользователи** — обновляются автоматически в течение 30 секунд
- **Токен бота** — обновляется автоматически в течение 30 секунд (бот перезапускает polling сам)

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и список команд |
| `/status` | Общий статус всех ботов |
| `/portfolio` | Сводка по портфелю |
| `/bots` | Список ботов с кнопками управления |
| `/bot <id>` | Детальная информация по боту |
| `/trades <id>` | Последние сделки бота |
| `/start <id>` | Запустить бота |
| `/stop <id>` | Остановить бота |
| `/restart <id>` | Перезапустить бота |
| `/risk` | Панель рисков |
| `/scoring` | Оценка стратегий |
| `/compare` | Сравнение B↔D↔L |
| `/agent` | Агентская стратегия — веса сигналов |
| `/help` | Справка |

Все команды поддерживают **inline-клавиатуру** — навигация по кнопкам без ввода текста.

## Безопасность

- Доступ к боту ограничен списком разрешённых Telegram ID
- Если список пуст — бот доступен всем, кто знает username бота
- Бот не хранит токен — он получает его из API FreqDash при запуске
- Все запросы к API идут с JWT-авторизацией
