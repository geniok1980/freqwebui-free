# 📊 FreqDash — Документация пользователя

**Версия 1.0** | [Telegram](https://t.me/geniok)

FreqDash — современная веб-панель и Telegram-бот для мониторинга и управления множеством торговых ботов [Freqtrade](https://www.freqtrade.io/) из единого интерфейса.

Интегрирована с **курсом обучения Freqtrade** — инструменты дашборда соответствуют методологии курса и помогают применить знания на практике.

## Возможности

| Функция | Описание |
|---------|----------|
| **Мониторинг ботов** | Единый дашборд для неограниченного количества ботов Freqtrade |
| **В реальном времени** | Данные через WebSocket |
| **Автообнаружение** | Автоматическое обнаружение ботов через Docker или файловую систему |
| **Портфельная аналитика** | Агрегированная статистика по всем ботам |
| **Лаборатория стратегий** | Запуск и мониторинг бэктестов, гипероптимизации и воркфлоу |
| **Финансовые данные** | Котировки криптовалют, акций, новости и макроэкономические индикаторы |
| **Агентская стратегия** | Динамические торговые агенты с адаптивными весами сигналов |
| **Селектор пар** | ML-оптимизация списка пар для торговли |
| **🤖 Telegram-бот** | Управление ботами и мониторинг через Telegram (inline-клавиатуры, форматированные карточки) |
| **✅ Pre-Launch Checklist** | Чек-лист запуска стратегии — 67 пунктов проверки (урок 22) |
| **📊 Scoring Dashboard** | Оценка стратегий по 4 группам метрик (уроки 6, 13) |
| **⚖️ Сравнение B↔D↔L** | Сравнение Backtest ↔ Dry-Run ↔ Live с допусками (уроки 16, 20, 23) |
| **🛡️ Risk Dashboard** | 4 уровня контроля рисков (уроки 14, 25) |
| **📓 Trading Journal** | Журнал торговли с шаблонами и оценкой сигналов (уроки 16, 24) |
| **📊 Редактор pairlist** | Изменение торговых пар прямо из интерфейса настройки бота |
| **Уведомления** | Оповещения о здоровье ботов, лимитах запросов и событиях |

## Архитектура

```
Веб-интерфейс (React, порт 15000)
       │ HTTP / WebSocket
Бэкенд API (FastAPI, порт 18001)
       │
├── PostgreSQL (основная БД + аналитика)
├── Redis (кэш)
└── Telegram Bot (aiogram, polling)
           │
           └── Telegram API
```

## Интерфейс

- **Выезжающий сайдбар** — открывается по кнопке-гамбургеру, в свёрнутом состоянии показывает только иконки
- **Группировка меню** — пункты сгруппированы по разделам (Торговля, Аналитика, Стратегии, Мониторинг) с раскрывающимися подменю
- **Тёмная/светлая тема** — переключается в настройках
