# 📈 Портфель

## Дашборд (`/`)

Главная страница собирает метрики со всех ботов в едином интерфейсе.

### Сводка портфеля

| Метрика | Описание |
|---------|----------|
| Всего ботов | Количество всех ботов в системе |
| В порядке | Боты, отвечающие на запросы |
| Недоступно | Боты, потерявшие связь |
| Гипероптимизация | Боты в режиме оптимизации параметров |
| Бэктест | Боты в режиме тестирования на истории |
| Торговых ботов | Активные боты, участвующие в портфеле |
| Общая прибыль | Прибыль портфеля в абсолютном значении |
| Прибыль в процентах | Относительная доходность портфеля |
| Общий баланс | Суммарный баланс всех ботов |
| Открытых позиций | Количество открытых сделок |
| Закрытых сделок | Количество завершённых сделок |
| Средний процент побед | Доля успешных сделок |
| Лучший бот | Наиболее прибыльный бот |
| Худший бот | Наименее прибыльный бот |

### Разбивка по биржам

Группировка метрик по биржам (Binance, Bybit, OKX и др.):

- Количество ботов на бирже
- Абсолютная прибыль
- Относительная прибыль в процентах
- Суммарный баланс
- Открытые позиции

### Разбивка по стратегиям

Группировка метрик по торговым стратегиям:

- Количество ботов на стратегии
- Абсолютная прибыль
- Относительная прибыль в процентах
- Открытые позиции
- Закрытые сделки
- Процент успешных сделок

### Лидеры

Рейтинг ботов, отсортированный по:
- проценту прибыли (по умолчанию)
- абсолютной прибыли
- проценту успешных сделок

Использует кэшированные данные из аналитической базы, поэтому метрики доступны даже когда боты отключены.

## Сравнение (`/compare`)

Сравнение ботов или стратегий бок о бок. Позволяет:
- Сравнить производительность нескольких ботов
- Сравнить метрики разных стратегий
- Визуализировать разницу в прибыли, проценте побед, просадке

## История (`/historic`)

Исторические данные по ботам, хранящиеся в аналитической базе (`bot_snapshots`):

- **Список ботов** — все боты, когда-либо имевшие данные, с отметкой активности
- **Последняя запись** — последние метрики для бота
- **Графики** — изменение метрик во времени (прибыль, процент побед, баланс, количество сделок)
- Активность бота определяется как наличие данных за последний час

Данные автоматически собираются сервисом сбора данных каждые 5 минут.
