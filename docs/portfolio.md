# 📈 Портфель

## Dashboard (`/`)

Главная страница собирает метрики со всех ботов в едином интерфейсе.

### Portfolio Summary

Сводка по всему портфелю:

| Метрика | Описание |
|---------|----------|
| Total Bots | Всего ботов |
| Healthy Bots | Здоровые боты |
| Unreachable Bots | Недоступные боты |
| Hyperopt Боты | Боты в режиме гипероптимизации |
| Backtest Боты | Боты в режиме бэктеста |
| Portfolio Боты | Активные торговые боты |
| Total Profit (Abs) | Абсолютная прибыль портфеля |
| Total Profit (%) | Относительная прибыль портфеля |
| Total Balance | Суммарный баланс |
| Open Positions | Открытые позиции |
| Closed Trades | Закрытые сделки |
| Avg Win Rate | Средний процент успешных сделок |
| Best Performer | Лучший бот по прибыли |
| Worst Performer | Худший бот по прибыли |

### Разбивка по биржам

Группировка метрик по биржам (Binance, Bybit, OKX и др.):

- Bot Count — количество ботов на бирже
- Profit Abs — абсолютная прибыль
- Profit % — относительная прибыль
- Balance — суммарный баланс
- Open Positions — открытые позиции

### Разбивка по стратегиям

Группировка метрик по торговым стратегиям:

- Bot Count — количество ботов на стратегии
- Profit Abs — абсолютная прибыль
- Profit % — относительная прибыль
- Open Positions — открытые позиции
- Closed Trades — закрытые сделки
- Win Rate — процент успешных сделок

### Top Performers

Рейтинг ботов отсортированный по:
- `profit_pct` (по умолчанию) — процент прибыли
- `profit_abs` — абсолютная прибыль
- `win_rate` — процент успешных сделок

Использует кэшированные данные из analytics DB, поэтому метрики доступны даже когда боты офлайн.

## Compare (`/compare`)

Сравнение ботов или стратегий бок-о-бок. Позволяет:
- Сравнить производительность нескольких ботов
- Сравнить метрики разных стратегий
- Визуализировать разницу в прибыли, win rate, drawdown

## Historic Data (`/historic`)

Исторические данные по ботам, хранящиеся в analytics database (`bot_snapshots`):

- **Список ботов** — все боты, когда-либо имевшие данные, с отметкой `is_active`
- **Latest Snapshot** — последняя метрика для бота
- **Time Series** — графики изменения метрик во времени (profit_all, winrate, balance, trade_count)
- Активность бота определяется как наличие данных за последний час

Данные автоматически собираются data-collector сервисом каждые 5 минут.
