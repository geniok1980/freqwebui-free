# 📊 FreqDash — User Documentation

**Version 1.0** | [GitHub](https://github.com/geniok1980/freqwebui-free) | [Telegram](https://t.me/geniok)

FreqDash — это современная веб-панель и мобильное приложение для мониторинга и управления множеством торговых ботов [Freqtrade](https://www.freqtrade.io/) из единого интерфейса.

## Возможности

| Функция | Описание |
|---------|----------|
| **Multi-Bot мониторинг** | Единый дашборд для неограниченного количества Freqtrade-ботов |
| **Real-Time** | Данные в реальном времени через WebSocket |
| **Auto-Discovery** | Автоматическое обнаружение ботов через Docker или файловую систему |
| **Портфельная аналитика** | Агрегированная статистика по всем ботам |
| **Strategy Lab** | Запуск и мониторинг бэктестов, гипероптимизации и воркфлоу |
| **Finance Data** | Котировки криптовалют, акций, новости и макроэкономические индикаторы |
| **Agent Strategy** | Динамические торговые агенты с адаптивными весами сигналов |
| **Pairlist Selector** | ML-оптимизация списка пар для торговли |
| **Alerts & Monitoring** | Уведомления о здоровье ботов, rate limits и событиях |
| **Mobile App** | Нативное приложение для iOS и Android |
