# 🤖 Agent Strategy (V8)

Agent Strategy — система динамических торговых агентов, которая адаптирует веса торговых сигналов в зависимости от рыночного режима.

## Как это работает

1. Система определяет текущий **рыночный режим** (bull, bear, ranging, volatile)
2. Для каждого режима используются свои **веса сигналов**
3. Агенты обучаются на исторических данных, оптимизируя веса
4. Результаты сохраняются в БД `financial_data`

Сигналы, используемые в агентах:

| Сигнал | Описание |
|--------|----------|
| Price Momentum | Моментум цены |
| Volume | Объём торгов |
| Sentiment | Рыночная тональность (новости + соцсети) |
| Macro | Макроэкономические данные |
| Orderbook | Данные книги ордеров |

### Управление агентом через Docker

Система предоставляет Docker-контроллер для управления контейнерами агентов:

- **Deploy container** — запуск нового Docker-контейнера агента
- **Stop / Start** — управление состоянием
- **View logs** — логи работы агента
- **Update** — обновление конфигурации агента
