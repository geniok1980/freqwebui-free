# 💰 Finance Data (V7)

Finance Data — интеграция внешних финансовых данных в FreqDash: криптовалюты, акции, новости и макроэкономические индикаторы.

## Сбор данных

Finance Data собираются автоматически через сервис **Finance Collectors** (включён по умолчанию). Отключить можно установкой `DISABLE_FINANCE_COLLECTORS=true` в `.env`.

Данные хранятся в отдельной БД `financial_data` (в том же PostgreSQL).

## Страница Finance Data (`/financedata`)

### Криптовалюты (Crypto Prices)

Таблица текущих цен криптовалют:

| Колонка | Описание |
|---------|----------|
| Coin | Монета (BTC, ETH, SOL...) |
| Name | Полное название |
| Price (USD) | Цена в долларах |
| Price (EUR) | Цена в евро |
| Price (BTC) | Цена в BTC |
| Market Cap | Рыночная капитализация |
| Volume 24h | Объём за 24 часа |
| Change 24h % | Изменение за 24 часа |
| Source | Источник данных |
